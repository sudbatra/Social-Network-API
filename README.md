# Social-Network-API

# Author: Sudhanshu Batra

## Descritpion: This application depicts the backend of a social media application. We can add new users, edit exisisting, get a list of all the users or specific user and delete users. We can also add new thoughts, update thoughts, get all the thoughts or get a specific thought by id.
